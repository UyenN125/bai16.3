Bước tiếp theo: Khởi tạo Git trong VsCode

Nó sẽ hiển thị một cửa sổ phía dưỡi màn hình, mặc định ở bên gốc phải là powershell kích vào dấu mũi tên để chuyển đổi Select default Profile chọn vào Git Bash Sau khi click sau quay lại giao diện phía dưới, nhấn vào biểu tượng thùng rác để xóa đi.
Và quay lại mở Terminal, chọn New Terminal, thì cửa sổ mới hiện lên sẽ vào Bash
